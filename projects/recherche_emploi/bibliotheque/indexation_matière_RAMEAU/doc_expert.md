# Indexation matière RAMEAU – Dossier expert complet

Objectif : maîtriser l’indexation matière RAMEAU comme une professionnelle expérimentée, afin de :

* décrire précisément le contenu d’un document ;
* produire des notices conformes aux normes bibliothéconomiques françaises ;
* permettre une recherche efficace dans les catalogues ;
* être à l’aise en entretien sur les aspects techniques.

---

## 1. Qu’est-ce que RAMEAU ?

RAMEAU = **Répertoire d’Autorité-Matière Encyclopédique et Alphabétique Unifié**.
C’est le **langage d’indexation contrôlé** utilisé dans les bibliothèques françaises (BNF, SUDOC, réseaux documentaires...).

RAMEAU fournit :

* des **vedettes-matière normalisées** (formes officielles des sujets),
* des **relations hiérarchiques** (générique/spécifique),
* des **relations associatives** (sujets proches),
* des **champs sémantiques structurés**.

Avantages :

* cohérence nationale,
* meilleure pertinence de recherche,
* élimination des doublons et synonymes.

---

## 2. But de l’indexation RAMEAU

L’indexation sert à :

* décrire **finement** les sujets abordés ;
* améliorer la **recherche par sujet** ;
* permettre la **navigation thématique** dans le catalogue.

Différence avec la Dewey :

* Dewey = classification = **une cote unique** pour ranger ;
* RAMEAU = indexation = **plusieurs vedettes** pour décrire.

---

## 3. Anatomie d’une vedette-matière RAMEAU

Une vedette RAMEAU peut comprendre :

* un **terme principal** (Nom commun, Nom propre, Groupe, Concept) ;
* des **subdivisions** :

  * `$x` subdivision de sujet (thème)
  * `$y` subdivision chronologique
  * `$z` subdivision géographique
  * `$v` subdivision de forme (documentaire)

Exemple :

```
606 ##$aFamille$xRelations parents-enfants$zFrance$y21e siècle$vOuvrages pour la jeunesse
```

---

## 4. Les types de vedettes RAMEAU

## 4.1. Vedettes « Nom commun »

Ce sont les sujets les plus fréquents.
Exemple :

* « Stress »
* « Révolution française »

## 4.2. Vedettes « Nom propre »

Pour personnes, collectivités, lieux.
Exemple :

* « Shakespeare, William »
* « France » (géographique)

## 4.3. Vedettes « Titre uniforme »

Pour œuvres littéraires, musicales, artistiques.
Exemple :

* « Bible. Nouveau Testament »

## 4.4. Vedettes complexes (avec subdivisions)

Exemples :

* « Écologie$xPhilosophie »
* « Cuisine végétarienne$vRecettes »

---

## 5. Processus professionnel d’indexation RAMEAU

Voici la méthode utilisée par les bibliothécaires experts.

## Étape 1 — Lire le document

* titre, sous-titre,
* résumé / 4ᵉ de couverture,
* introduction.

## Étape 2 — Identifier les **idées principales**

Un document peut avoir plusieurs thèmes → choisir 2 à 5 sujets pertinents.

## Étape 3 — Trouver la vedette RAMEAU correspondante

Utiliser :

* le site RAMEAU (BNF),
* un catalogue professionnel (SUDOC),
* un SIGB doté d’autorités.

## Étape 4 — Choisir la forme **autorisée**

Ne jamais inventer un terme : utiliser la **forme officielle**.

## Étape 5 — Ajouter des subdivisions

Selon le document :

* thème secondaire → `$x`
* période → `$y`
* zone → `$z`
* forme → `$v`

## Étape 6 — Vérifier la cohérence

* vérifier les renvois « voir » / « employer pour »,
* s’assurer que la vedette est bien adaptée au contenu réel.

## 6. Exemple complet d’indexation pour différents types de documents

## 6.1. Document : *Guide de gestion du stress au travail*

Sujets identifiés : stress, conditions de travail, psychologie.

Vedettes RAMEAU :

```
606 ##$aStress (psychologie)
606 ##$aTravail$xConditions de travail
606 ##$aBien-être au travail
```

## 6.2. Document : *Histoire du Japon médiéval*

```
606 ##$aJapon$xHistoire$yMoyen Âge
```

## 6.3. Document : *Cuisine végétarienne pour débutants*

```
606 ##$aCuisine végétarienne$vRecettes
```

## 6.4. Document : *Introduction à l’écologie politique*

```
606 ##$aÉcologie$xPolitique et gouvernement
```

## 7. Règles essentielles à maîtriser comme une experte

## 7.1. Toujours utiliser la forme autorisée

Exemple :

* « Stress » n’est pas autorisé.
* « Stress (psychologie) » est la forme correcte.

## 7.2. Appliquer la logique du général → spécifique

Document sur « abeilles » :

* Vedette principale : « Abeilles »
* Thème secondaire : « Apiculture » si relevant

## 7.3. Subdivisions de forme ($v)

Appliquées au **type de document** :

* Anthologies
* Bibliographies
* Recettes
* Guides pratiques
* Cartes

## 7.4. Subdivisions géographiques ($z)

Utiliser **la forme RAMEAU du lieu**, pas l’usage courant.

## 7.5. Hiérarchie sémantique

Utiliser les termes les plus **spécifiques**.

## 9. Expressions professionnelles à utiliser en entretien

* « J’identifie les concepts principaux du document et je sélectionne les vedettes RAMEAU les plus pertinentes. »
* « Je maîtrise les subdivisions de forme, de temps, de lieu et de sujet. »
* « Je vérifie toujours la cohérence des vedettes avec les autorités RAMEAU. »
* « Je sais indexer de manière précise et homogène pour optimiser la recherche documentaire. »

### Autres formulations possibles en entretien

* « En indexation RAMEAU, j’identifie d’abord les idées principales du document, puis je choisis les vedettes autorisées les plus pertinentes avec les subdivisions adaptées ($x, $y, $z, $v). »
* « Mon objectif est que la recherche par sujet dans le catalogue soit claire et cohérente pour les usagers, avec des vedettes homogènes d’un document à l’autre. »

---

## 10. Pyramide RAMEAU — résumé visuel

1. **Terme principal** (obligatoire)
2. **$x — Sujet secondaire** (facultatif)
3. **$z — Lieu** (facultatif)
4. **$y — Période** (facultatif)
5. **$v — Forme du document** (facultatif)

Exemple final :

```
606 ##$aFemmes$xDroits$zEurope$y21e siècle
```
